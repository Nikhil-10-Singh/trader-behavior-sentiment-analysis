# Trader Behavior vs Market Sentiment Analysis

## Objective
Analyze how market sentiment (Fear vs Greed) impacts trader performance and behavior.

## Tools Used
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## Methodology
- Cleaned and merged trader + sentiment data
- Created metrics: PnL, win rate, trade size, trade frequency
- Segmented traders into high vs low size groups

## Key Insights
- Sentiment has minimal impact on profitability
- Traders take larger risks during Fear
- Trading activity increases during Greed
- Trade size is the biggest driver of profit

## Strategy Recommendations
- Reduce trade size during Fear
- Increase activity during Greed with risk control
- Focus on position sizing over sentiment
